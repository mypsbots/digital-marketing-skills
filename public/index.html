Digital Marketing Skills — MCP server

Model Context Protocol server exposing a library of digital-marketing skills, prompts and tools.

Endpoint

Streamable HTTP MCP endpoint:

POST /api/mcp

When DM_MCP_HTTP_AUTH_TOKEN is configured, send Authorization: Bearer <token> with every request.

Connect a client

Add this URL as a custom MCP connector (ChatGPT, Claude web, etc.). See the repository for full setup docs.
