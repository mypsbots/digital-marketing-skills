# Digital Marketing Skills — MCP Platform

A Model Context Protocol (MCP) server and a curated library of digital-marketing skills that lets
AI assistants discover and apply industry-standard marketing methodology safely.

- **175 skills** across 31 categories, each a rich `SKILL.md` (methodology, decision rules,
  checklists, examples, KPIs).
- **MCP server** exposing resources, 30 prompts and controlled tools.
- **8 approval-gated workflows** and per-industry playbooks.
- **Deterministic engines** for metrics, policy, approvals and audit.
- **Optional Python utilities** for metrics and experiment statistics.

## Principles

1. **Honesty over hype** — no guaranteed results; assumptions and ranges are explicit.
2. **Privacy by design** — data minimisation, consent checks, PII redaction.
3. **Human-in-the-loop** — risky, costly or side-effecting actions require approval.
4. **Determinism where it matters** — metric maths are pure and auditable.
5. **Safe defaults** — read-only, dry-run, mock connectors unless explicitly changed.
